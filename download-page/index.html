Translator

划词翻译 + 截图翻译，macOS 原生体验

macOS 版本 DMG 安装包 · 791 KB

需要 macOS 15.0 (Sequoia) 或更高版本

⚠️ 提示「无法验证开发者」？

应用没有苹果签名，不影响使用。按以下步骤安装：

1 打开 DMG，将 TranslatorApp 拖到 Applications
2 打开「系统设置」→「隐私与安全性」
3 滚动到底部，点击「仍要打开」

或者用终端命令：

Terminal
$ xattr -cr /Applications/TranslatorApp.app

点击复制命令

  • 划词翻译：选中文本 + ⌥T 即时翻译
  • 截图翻译：⌘⇧S 截取区域 OCR 翻译
  • 智能粘贴：截图复制自动适配图片/路径
  • 单词本：收藏单词，支持导出 CSV/JSON
  • 完全本地：使用 Apple 翻译引擎，隐私安全
